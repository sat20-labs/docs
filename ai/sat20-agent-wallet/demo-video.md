# SAT20 Agent Wallet 1 分钟演示视频

这是 SAT20 Agent Wallet / STP 的对外演示视频入口。

视频使用真实的 BTC testnet4 与 SatoshiNet testnet 演练素材，核心展示的是：

1. 用户通过钱包控制助记词和签名授权。
2. STP 通道把资产放入可验证的 2-of-2 安全边界。
3. 解锁推进提交高度后，旧承诺状态会变成可惩罚状态。
4. 当旧承诺被广播到 BTC L1 后，用户钱包可以广播惩罚交易，把资产扫回自己控制。
