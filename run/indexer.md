# L1 / L2 Indexer

Indexer 是聪网资产事实层。L1 Indexer 负责 BTC 主网多协议资产事实，L2 Indexer 负责聪网交易、UTXO、通道、合约和跨层状态。

## 运行价值

1. 为钱包和交易平台提供资产查询。
2. 为 Explorer 提供可视化数据。
3. 为 STP 和 Agent 提供跨层证据。
4. 为 DEX、DAO 和合约应用提供状态查询。
5. 通过多方运行降低单点依赖。

## 需要补充

1. L1 Indexer 部署要求。
2. L2 Indexer 部署要求。
3. 数据备份和 reorg 处理。
4. API 稳定性和错误码。
5. 多 indexer 交叉验证方式。
