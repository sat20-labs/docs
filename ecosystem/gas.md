# 网络费用与 GAS

GAS 是聪网中用于合约执行、交易处理和网络资源定价的费用资产。它的意义来自真实应用、真实交易和真实资源消耗。

## GAS 用于什么

GAS 服务于：

1. 合约执行。
2. 交易处理。
3. 网络资源定价。
4. 防止滥用。
5. 节点和生态激励。

如果聪网没有真实应用、真实交易和真实合约执行，GAS 就没有稳定的使用基础。因此 GAS 的公开表达聚焦于用途、消耗、费用模型、节点激励、用户体验和治理边界。

## 建设者关心的问题

| 问题 | 当前答案 |
| --- | --- |
| 如何获得测试 GAS | 通过测试网工具、Faucet 或社区测试分发流程获得 |
| 如何估算合约成本 | 以合约执行、交易大小、状态访问和当前费用规则为基础估算 |
| 节点如何获得费用 | 节点通过处理交易、出块和网络服务参与费用分配 |
| 社区能否赞助用户 GAS | 社区 DApp 可以设计补贴、代付或引导式领取流程 |
| DApp 如何降低首次使用门槛 | 钱包引导、测试网演示、Gas 赞助和清晰的费用预估 |
| GAS 与节点质押的关系 | 质押、费用和节点角色会随网络治理和协议实现逐步公开 |

## GAS 所依赖的基础

1. Indexer 能统一表达 BTC L1 多协议资产事实。
2. STP 能让用户安全进入和退出聪网。
3. SatoshiNet 能承载合约执行和应用交易。
4. 开发者有理由部署合约。
5. 应用产生持续交易和资源消耗。

Indexer 和 STP 是资产入口基础，智能合约是使用需求，GAS 是网络资源的费用表达。

## 风险边界

GAS 聚焦于：

1. 用途。
2. 消耗。
3. 费用模型。
4. 生态激励。
5. 治理和风险。

更完整的 GAS 协议和经济模型会随智能合约实现逐步补充。
