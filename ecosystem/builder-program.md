# Builder Program

Builder Program 用于吸引和支持聪网生态建设者。它服务于长期公共品，而不是短期活动。

## 支持方向

| 方向 | 示例 |
| --- | --- |
| 基础设施 | 钱包、Indexer、Explorer、SDK、监控 |
| STP 客户端 | 多语言 adapter、PWA bridge、测试工具 |
| 智能合约 | AMM、稳定币、支付、资产市场 |
| AI Agent | SAT20 Agent Wallet skill、钱包助手、合约助手、安全验证 |
| 教育内容 | 教程、视频、可复核测试网演练 |
| 研究 | RSMC、GAS 经济、合约安全、跨层状态 |

## 评估标准

1. 是否增强用户资产控制权。
2. 是否降低开发者接入成本。
3. 是否提升聪网资产流动性。
4. 是否形成可复用公共品。
5. 是否有测试网或主网可验证结果。

## 文档要求

每个生态项目资料包含：

1. 项目目标。
2. 当前状态。
3. 使用方式。
4. 风险边界。
5. 测试网或主网证据。
6. 与 STP、GAS、合约或 Agent 的关系。
7. 是否能提供可复核的资产事实、交易证据或状态证明。
