# Ecosystem：生态建设

聪网生态需要开发者、资产方、交易平台、钱包、Indexer、做市商、AI Agent 构建者、研究者和社区成员共同参与。

文档的目标不是只解释协议，而是让每一类建设者知道自己可以在哪里创造价值。

## 为什么在聪网上构建

1. 资产来自比特币主网。
2. Indexer 将多协议资产统一表达为可验证的资产事实。
3. STP 提供可验证的用户控制权。
4. 聪网提供更低成本、更快确认和智能合约环境。
5. GAS 为合约执行和生态激励提供经济入口。
6. AI Agent 可以降低复杂协议的使用门槛。

## 建设者地图

| 建设者 | 机会 |
| --- | --- |
| 钱包团队 | PWA adapter、移动钱包、硬件钱包集成 |
| 交易平台 | BTC L1 / 聪网充值提现、资产市场 |
| 开发者 | AMM、稳定币、支付、资产发行、游戏、DePIN |
| Indexer / Explorer | L1/L2 资产、通道、合约和交易数据 |
| AI Agent 团队 | 自动化钱包、合约助手、安全验证助手 |
| 资产社区 | Runes、BRC20、ORDX 等资产进入聪网流动 |
| 投资机构 | GAS 生态、基础设施和应用机会 |

## 生态增长飞轮

1. Indexer 统一表达 BTC L1 多协议资产。
2. STP 让 BTC 资产安全进入聪网。
3. 智能合约创造使用场景。
4. GAS 为网络资源定价。
5. 钱包和 Agent 降低使用门槛。
6. 交易平台、Indexer 和 Explorer 提供流动性与可见性。
7. 更多资产和应用进入聪网。
