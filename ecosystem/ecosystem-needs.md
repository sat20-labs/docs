# Ecosystem Needs：当前生态需求

本文列出聪网当前最需要的建设者和合作方向。它用于帮助外部团队判断自己可以在哪里创造价值。

## 当前优先需求

| 方向 | 需要什么 | 适合谁 |
| --- | --- | --- |
| 社区 DEX 试点 | 选择 BTC 资产社区，完成测试网 DEX / DAO 原型 | 资产社区、开发团队、运营团队 |
| EVM 工具链 | Solidity 部署、调试、RPC、SDK、示例合约 | EVM 开发者、工具链团队 |
| 经典合约迁移 | AMM、限价单、支付、稳定币、Launchpad 模板 | 合约开发者 |
| 钱包集成 | PWA adapter、移动钱包、硬件钱包、Wallet SDK | 钱包团队 |
| Indexer / Explorer | L1/L2 资产、交易、合约、通道和跨层证据展示 | 数据服务和基础设施团队 |
| 节点运营 | Core Node、RPC、监控、稳定运行 | 基础设施团队 |
| 做市和流动性 | AMM 池、限价单市场、跨社区流动性 | 做市商、资产社区 |
| 安全审计 | STP、合约、钱包 adapter、Indexer 证据链 | 安全团队和研究者 |
| AI Agent | 钱包安全助手、Community Builder Agent、合约操作助手 | Agent 开发者 |
| 内容和社区 | 英文内容、教程、视频、区域社区 | 内容团队、社区运营者 |

## SAT20 Labs 可以提供的支持

1. 技术方案讨论。
2. 测试网环境和验证路径。
3. 开源代码和源码入口。
4. 钱包、Indexer、Explorer、合约和 STP 的接入建议。
5. 文档、教程和联合发布支持。
6. 在资源允许时，提供开发资助或合作开发支持。

## 如何参与

1. 选择一个方向。
2. 准备你的团队、项目目标、需要的模块和预期时间表。
3. 阅读 [Builder Program](builder-program.md)。
4. 通过 GitHub、X 或后续公开申请表联系 SAT20 Labs。
5. 先在测试网完成可验证结果，再进入公开展示。
