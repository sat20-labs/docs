# 支持协议开发

长期协议开发需要透明、可持续的支持结构。支持协议开发不等同于购买 GAS，也不代表价格收益、网络利润或治理权承诺。

## 支持方式

| 方式 | 用途 |
| --- | --- |
| BTC 捐赠 | 支持开源维护、文档、测试网和公共服务 |
| 开源 Grant | 支持明确公共品里程碑 |
| 企业赞助 | 支持 EVM Runtime、安全审计、Wallet SDK、公共 Indexer / Explorer / RPC |
| 技术服务 | Community Stack 部署、钱包或交易平台集成、托管节点与 API |
| 基础设施资源 | 服务器、带宽、监控、节点和 Explorer |
| 文档与翻译 | 支持中文、英文和区域社区材料 |

## 边界声明

1. 支持协议开发不等同于购买 GAS。
2. 不承诺支持方获得价格收益或网络利润。
3. 未来基金会成立前，不以基金会名义接收或管理资金。
4. 正式资金页面应列明接收实体、用途、预算、负责人和交付物。
