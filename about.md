# 关于 SAT20 Labs

SAT20 Labs 是聪网核心协议与开源基础设施的建设团队。我们研发并维护 Indexer、STP、SatoshiNet、钱包、智能合约、DEX、DAO、Explorer 和开发者工具，并与 BTC 社区、开发者、节点运营者、钱包、交易平台、做市商和 AI Agent 团队共同建设开放生态。

SAT20 Labs 的工作重点包括：

1. 维护 BTC L1 多协议资产的统一 indexer 能力。
2. 推进 STP 通道协议、资产跨层流转和用户退出安全。
3. 建设 SatoshiNet 节点、合约运行环境、GAS 费用模型和 L2 indexer。
4. 提供钱包 SDK、PWA Wallet、Agent adapter 和开发者工具。
5. 帮助 BTC 社区在测试网和后续主网中搭建 DEX、DAO、Explorer、钱包入口和社区基础设施。

更多当前团队职责见 [SAT20 Labs](governance-support/sat20-labs.md)。

## 未来基金会

聪网未来计划成立独立的非营利基金会，负责公共协议、开源资助、安全审计、生态 Grant 和透明治理。基金会目前尚未成立，正式结构、理事、资金和治理规则将在完成法律与治理设计后公开。

更多边界见 [未来基金会](governance-support/future-foundation.md)。

## 社区可以自主做什么

SAT20 / 聪网的生态目标是让 BTC 社区拥有自己的基础设施。社区可以自主运营品牌、域名、节点、资产规则、流动性、DAO 参数、社区基金和用户支持。SAT20 Labs 提供开源代码、协议文档、测试网、工具和技术接入支持，但社区的链上签名、治理决策和运营规则应由社区自己负责。

## 贡献方式

你可以通过以下方式参与：

1. 运行 Core Node、Indexer、Explorer 或公共 RPC。
2. 构建钱包、DEX、DAO、Launchpad、合约和数据工具。
3. 集成 SAT20 Agent Wallet 或构建 Community Builder Agent。
4. 提交文档、教程、翻译、演示视频和测试网证据。
5. 参与安全审计、协议研究和故障恢复流程设计。
6. 申请 [Builder Program](ecosystem/builder-program.md)。

## 安全问题

如果你发现 STP、钱包、合约、Indexer、节点或文档中的安全问题，请参考 [安全披露](governance-support/security-disclosure.md)。在正式安全披露渠道建立前，可以通过 GitHub 或官方公开渠道联系 SAT20 Labs，并尽量提供可复现步骤、交易、日志、版本和影响范围。
